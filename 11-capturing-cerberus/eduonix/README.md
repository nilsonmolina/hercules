# [Eduonix Socket Programming Tutorial in C](https://youtu.be/LtXEMwSG5-8)

## **Commands to Run**   
1. **Create Binaries**
    ```bash
    $ make tcp_client
    cc     tcp_client.c   -o tcp_client
    $ make tcp_server
    cc     tcp_server.c   -o tcp_server
    ```

1. **Run Server**
    ```bash
    $ ./tcp_server
    ```

1. **Run Client**
    ```bash
    $ ./tcp_client
    The server sent the data:
    - You have reached the server!
    ```
